# 🕋 مشروع مناسك الحج | Hajj Manasik Guide

![Hajj Header](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/aqua.png)

> **دليل رقمي تفاعلي متكامل لأداء فريضة الحج، يجمع بين روحانية الشعيرة وعصرية التصميم.**

## ✨ المميزات الرئيسية (Core Features)

* **🎨 واجهة مستخدم فاخرة:** تصميم "Dark Mode" مع لمسات ذهبية تعكس جلال الكعبة المشرفة.
* **🌍 دعم ثنائي اللغة:** تبديل فوري بين العربية والإنجليزية بنقرة واحدة مع تغيير كامل لاتجاه الصفحة (RTL/LTR).
* **📱 تصميم متجاوب تماماً:** متوافق مع كافة أحجام الشاشات (Mobile, Tablet, Desktop).
* **🎞️ أنيميشن واقعي:** استخدام CSS Keyframes لخلق حركات انسيابية تذكرنا بهدوء الطواف والسعي.
* **📜 مقتبسات قرآنية:** عرض آيات من سورة البقرة بتنسيق فني يتوسط الصفحة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

| التقنية | الاستخدام |
| :--- | :--- |
| **HTML5** | الهيكلية المتقدمة والعناصر الدلالية (Semantic UI). |
| **CSS3** | المتغيرات (Variables)، الأنيميشن، وفلاتر الزجاج (Glassmorphism). |
| **JavaScript** | نظام تبديل اللغات وإدارة حالة الصفحة (DOM Manipulation). |
| **SVG** | أيقونات مخصصة (أيقونة الكعبة المشرفة) لضمان أعلى جودة وضوح. |
