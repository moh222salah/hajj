## 🏛️ Project Vision
**Hajj Manasik** is a high-fidelity digital gateway designed to provide a serene and spiritually uplifting experience for pilgrims. This platform bridges the gap between **Sacred Traditions** and **Modern Software Craftsmanship**, utilizing a cinematic "Dark Gold" aesthetic that mirrors the majesty of the Holy Kaaba.

---

## ✨ Premium Features & Animation Logic

### 🌙 Cinematic Visual Flow
The interface is engineered with a **"Peaceful Motion"** philosophy. Every element transition is designed to mimic the calm and rhythmic movement of *Tawaf*, using:
- **Smooth Fade-ins:** Orchestrated CSS keyframes for a graceful entry.
- **Backdrop Blur (Glassmorphism):** Creating depth and focus on sacred texts.
- **Dynamic Glow:** A subtle golden aura around primary UI elements.

### 🌍 Global Accessibility (i18n)
A robust, zero-latency language engine that supports:
- **Arabic (RTL):** Preserving the traditional calligraphy and linguistic soul.
- **English (LTR):** Providing clear, modern guidance for a global audience.
- *Instant directional layout flipping without page reload.*
